# Demo Guide

## Installation
1. Copy this folder to `wp-content/plugins/`
2. Activate in WordPress admin → Plugins
3. Demo content is created automatically on activation

## What Happens on Activation
- Sample data is seeded into the database
- Plugin settings are initialized with defaults
- Admin menu items appear in the WordPress sidebar

## Testing
- Visit the WordPress admin dashboard
- Check the new menu items added by this plugin
- Use the REST API endpoints at `/wp-json/` to test programmatically
